## About

**Solvion-Launcher** is a modified fork of Mythic.

Modified and maintained by **emr09**.

## Features

* [x] Windows Game Support (DX9-12) (64-bit)
* [ ] Steam Support (Windows version of Steam can still be manually downloaded)
* [x] Epic Games Support
* [x] Manual Game Imports
* [x] Game management
* [x] Discord Integration

## Download

Download releases from this repository.

* Please note this launcher currently supports macOS 14 (Sonoma) and above.

## Dependencies

This project uses the following Swift dependencies:

* Sparkle — Software update framework for macOS
* SwiftyJSON — JSON handling for Swift
* SemanticVersion — Semantic version parsing
* ColorfulX — Metal gradient rendering
* Glur — SwiftUI progressive blur library
* WhatsNewKit — Display new features in app
* firebase-ios-sdk — Firebase SDK for Apple platforms
* SwiftUI-Shimmer — Shimmer UI effect
* DockProgress — Dock icon progress display
